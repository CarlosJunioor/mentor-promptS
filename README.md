# mentor-prompts

An **unofficial**, community-made prompt generator for
[OutSystems Mentor](https://www.outsystems.com/low-code-platform/mentor-ai-app-generation/),
built with Claude Code.

It helps developers using the free OutSystems Personal Environment (ODC)
learn how to write effective prompts for Mentor's AI app generation —
covering data models, roles, screens, and iteration prompts.
